# 🏢 Business Consulting & Digital Agency Platform

A premium, modern, and fully responsive website designed for business consulting firms and digital agencies. Built using **React**, **Vite**, and **Tailwind CSS**, this platform showcases interactive layouts, clean typography, and a seamless user experience.

---

## 🚀 Live Demo

Check out the live deployment here:  
🔗 **[Live Preview Link](https://react-frontend-devel-lxyx.bolt.host/)**

---

## ✨ Features

- **Modern UI/UX:** Clean, elegant, and corporate design tailored for modern businesses.
- **Fully Responsive:** Optimised perfectly for Mobile, Tablet, and Desktop screens.
- **Dynamic Pricing Table:** Toggle-based pricing cards for different business scales.
- **Interactive FAQ Section:** Accordion-style Frequently Asked Questions.
- **Client Testimonials:** Dedicated section displaying social proof and reviews.
- **Case Studies & Insights:** Grid-based presentation of featured works and recent blog articles.

---

## 🛠️ Tech Stack

- **Frontend Framework:** React.js (via Vite)
- **Styling:** Tailwind CSS & daisyUI (Component Library)
- **Icons:** Lucide React / React Icons
- **Deployment:** Hosted on Bolt.host / Netlify
